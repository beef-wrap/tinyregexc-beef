# About

A Beef wrapper for [mylib](https://github.com/upstream_owner/upstream_reo), a upstream_description.

# Building static libs

```sh
zig build --prefix libs
```

# Running the example

Clone the repo, and open up the workspace in the Beef IDE, and run.
